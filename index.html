Uma aventura desenhada

Propósito

A fim de testar o que venho aprendendo nesse tempo de curso, resolvi desenvolver um site básico com o propósito de ter uma pequena tentativa de "animação" envolvendo Javascript. Vão ser só imagens estáticas que vão para um lugar ou outro, mas a intenção mesmo é só criar qualquer coisa.

Instrumentos usados

Para a criação desse site estaremos usando o Html para fazer o corpo de tudo, CSS para dar cores e o Javascript para que tudo circule corretamente.

Html

Nada muito fora do comum, Html serve justamente para formarmos a estrutura do site e criarmos textos para complementá-lo. Usaremos muitas tags semânticas como: articles, main, header. Mas isso apenas importa para os mecanismos de busca.

CSS

Aqui serve para montarmos o estilo de um site. Sem ele os sites de hoje pareceriam apenas textos em um fundo branco. Faremos coisas básicas como mudança do background, das fontes e deixar visualmente agradável

Javascript

A parte mais complicada da criação de um site, por assim dizer. Aqui é onde decidimos tudo que o site pode ou não poderá fazer, uma ferramenta poderosa mas um pouco complicado de se pegar no inicio. Usaremos os artíficios básicos de uma programação como: funções, variáveis, mas principalmente comandos if e elses, que é o que servirá para criarmos a "animação".
